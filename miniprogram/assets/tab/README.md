# tabBar 图标占位

此目录需放置 10 个 PNG 图标文件（推荐 81×81px，小于 40KB）：

- trends.png / trends-active.png
- ai.png / ai-active.png
- data.png / data-active.png
- growth.png / growth-active.png
- profile.png / profile-active.png

**颜色规范：**
- 未选中：`#5F6368`
- 选中：`#1A73E8`

**图标建议来源：** Material Symbols Outlined / Feather Icons
- 趋势: trending_up
- AI: psychology
- 数据: bar_chart
- 成长: auto_graph
- 我的: person

可先用任意图标占位让开发者工具不报错，后续再替换为正式设计稿。
